Medidas Relativas

6) Por que usar medidas relativas?

Medidas relativas ajudam o site a se adaptar melhor a diferentes telas e tamanhos de texto. Elas são mais eficientes do que medidas fixas como px.

% — Porcentagem

Relativa ao elemento pai. Se o pai tem 800px de largura, width: 50% resulta em 400px. 

.caixa {
  width: 80%;
}
80%
50%
30%

rem — Root em

Relativa ao tamanho de fonte do elemento raiz <html>. Por padrão, 1rem = 16px. 

h1 {
  font-size: 2rem; /* 32px */
}

p {
  font-size: 1rem; /* 16px */
}

Texto 1rem

Texto 0.8rem

vw — Viewport Width

Relativa à largura da janela do navegador. 1vw = 1% da largura da tela. 

.titulo {
  font-size: 5vw;
}

Este texto usa 3vw — cresce com a tela!

vh — Viewport Height

Relativa à altura da janela do navegador. 100vh = altura total da tela visível. 

.hero {
  height: 100vh;
}

Esta caixa tem 20vh de altura

Medidas fixas vs relativas

❌ Medidas fixas (px)

.caixa {
  width: 600px;
  font-size: 24px;
  height: 400px;
}

Problemas: não se adapta a telas menores ou maiores. O layout pode quebrar.

✅ Medidas relativas

.caixa {
  width: 80%;
  font-size: 1.5rem;
  height: 50vh;
}

Vantagem: se adapta automaticamente a qualquer tamanho de tela.
