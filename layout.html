Flexbox e Grid

8) Diferença entre Flexbox e Grid

Tanto Flexbox quanto Grid são formas de organizar elementos na tela com CSS. A diferença está em como cada um trabalha: Flexbox em uma direção só, Grid em linhas e colunas ao mesmo tempo.

Flexbox — uma direção

flex-direction

Define se os itens ficam em linha ou em coluna.

.container {
  display: flex;
  flex-direction: row; /* padrão */
}

.container {
  display: flex;
  flex-direction: column;
}

gap e flex-wrap

gap define o espaço entre os itens. flex-wrap permite que os itens quebrem linha quando não cabem. 

.container {
  display: flex;
  gap: 1rem;
  flex-wrap: wrap;
}

9) Quando usar Flexbox?

Use Flexbox quando precisar organizar elementos em uma direção: 

✔ Menus de navegação
✔ Botões lado a lado
✔ Centralizar um elemento na tela
✔ Barras com logo + links
✔ Formulários em coluna

Grid — linhas e colunas

grid-template-columns

Define quantas colunas o grid terá e qual o tamanho de cada uma. 

/* 3 colunas iguais */
grid-template-columns:
  repeat(3, 1fr);

/* colunas de tamanhos fixos */
grid-template-columns:
  200px 1fr 200px;

/* adapta automaticamente */
grid-template-columns:
  repeat(auto-fit,
  minmax(150px, 1fr));

grid-template-rows

Define a altura de cada linha do grid.

.container {
  display: grid;
  grid-template-columns:
    repeat(2, 1fr);
  grid-template-rows:
    80px 120px;

gap no Grid

Define o espaço entre as linhas e colunas do grid.

.container {
  display: grid;
  grid-template-columns:
    repeat(3, 1fr);

  /* gap igual em todos os lados */
  gap: 1rem;

  /* gap diferente: linha coluna */
  gap: 2rem 0.5rem;
}

9) Quando usar Grid?

Use Grid quando precisar organizar elementos em duas direções: 

✔ Layout de página inteira
✔ Galerias de fotos
✔ Painéis com colunas e linhas
✔ Sistemas de cards complexos

Resumo: Flexbox vs Grid

Flexbox

display: flex
flex-direction: row | column
justify-content: center | space-between...
align-items: center | flex-start...
gap: 1rem
flex-wrap: wrap

→ Melhor para 1 direção

Grid

display: grid
grid-template-columns: repeat(3, 1fr)
grid-template-rows: 100px 200px
gap: 1rem

→ Melhor para 2 direções

@todos os direitos reservados
