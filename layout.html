Flexbox e Grid

8) Diferença entre Flexbox e Grid

Tanto Flexbox quanto Grid são formas de organizar elementos na tela com CSS. A diferença está em como cada um trabalha: Flexbox em uma direção só, Grid em linhas e colunas ao mesmo tempo.

Flexbox — uma direção

O que é?

O Flexbox organiza elementos em linha ou coluna. Ele é muito usado para menus, barras de navegação e alinhar itens lado a lado. 

.menu {
  display: flex;
  gap: 1rem;
}

/* coluna */
.formulario {
  display: flex;
  flex-direction: column;
}

9) Quando usar Flexbox?

Use Flexbox quando precisar organizar elementos em uma direção:

✔ Menus de navegação
✔ Botões lado a lado
✔ Centralizar um elemento
✔ Barras com logo + links

Grid — linhas e colunas

O que é?

O Grid trabalha com linhas e colunas ao mesmo tempo. É ideal para estruturas de páginas completas, galerias de imagens e sistemas de cards. 

.galeria {
  display: grid;
  grid-template-columns:
    repeat(3, 1fr);
  gap: 1rem;
}

9) Quando usar Grid?

Use Grid quando precisar organizar elementos em duas direções:

✔ Layout de página inteira
✔ Galerias de fotos
✔ Painéis com colunas e linhas
✔ Sistemas de cards complexos

Resumo: Flexbox vs Grid

Flexbox

Direção: 1D (linha OU coluna)
Uso: menus, barras, alinhamento
Propriedade: display: flex

Grid

Direção: 2D (linhas E colunas)
Uso: layouts, galerias, cards
Propriedade: display: grid
