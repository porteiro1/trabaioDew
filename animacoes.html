Animações CSS

10) Animações com CSS

O CSS permite criar animações sem precisar de JavaScript. As duas formas principais são: transition (para efeitos no hover) e @keyframes + animation (para animações automáticas).

transition

Cria uma transição suave quando uma propriedade muda de valor. Muito usado com :hover. 

.botao {
  background: gray;
  transition: background 0.3s;
}

.botao:hover {
  background: #444;
}

transform

Muda a forma ou posição de um elemento visualmente. Pode ser usado junto com transition. 

.caixa:hover {
  transform: scale(1.1);
  transform: rotate(5deg);
  transform: translateX(10px);
}

@keyframes

Define os estados de uma animação. Você escolhe o que acontece no início (from / 0%) e no fim (to / 100%). 

@keyframes piscar {
  from { opacity: 1; }
  to   { opacity: 0; }
}

animation

Aplica o @keyframes a um elemento, definindo duração, repetição e estilo. 

.caixa {
  animation: piscar
             1s
             infinite
             alternate;
}

Demonstrações de animação

Pulsando

@keyframes pulsar {
  0%   { transform: scale(1); }
  50%  { transform: scale(1.2); }
  100% { transform: scale(1); }
}

Deslizando

@keyframes deslizar {
  from { transform: translateX(0); }
  to   { transform: translateX(60px); }
}

Girando

@keyframes girar {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

Piscando

@keyframes piscar {
  from { opacity: 1; }
  to   { opacity: 0; }
}

Propriedades principais

transition

transition: propriedade
            duração
            timing;

Exemplo:
transition: background 0.3s ease;

animation

animation: nome
           duração
           repetição
           direção;

Exemplo:
animation: pulsar 1s infinite;
